# awp explore

Enter explore mode — interactive thinking partner for exploring ideas and clarifying requirements before proposing.

No code changes, only investigation and discussion. Help the user think through:
- What problem they're solving
- Possible approaches and trade-offs
- Requirements and constraints
- Scope and boundaries

When the user is ready, suggest using `/awp-propose` to formalize the idea.
